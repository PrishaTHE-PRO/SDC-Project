# **App Name**: BadgerDeals

## Core Features:

- Email Verification: Verify user accounts using @wisc.edu student email addresses.
- Listing Creation: Allow logged-in users to create and post new listings with associated tags.
- Messaging System: Enable users to send direct messages to each other regarding listings.
- Personalized Feed: Display listings in a social media-style feed, prioritizing listings with tags matching the user's past interactions and message history. Use 'tool' to determine listing order.
- Tagging System: Allow listings to be tagged with keywords for categorization and search.
- Trending Listings: Showcase the most recently viewed listings on a dedicated 'Trending' page.
